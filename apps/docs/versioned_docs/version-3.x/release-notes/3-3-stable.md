# 3.3 Stable

`3.3` is the first stable line documented in this site.

## What landed

- Backdrop component support
- More stable overlay behavior and reduced JS coordination during push transitions
- Expanded snap-point behavior, including lock and programmatic control improvements across the prerelease train
- Better nested-route and scroll-hand-off behavior

## Why it matters

`3.3` is the line where the package stopped being "custom transitions only" and became a more complete gesture and sheet system.

If you are onboarding on stable today, this is the line the docs optimize for.
