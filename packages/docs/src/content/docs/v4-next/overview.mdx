This preview currently has a single page on purpose. It gives you a real version target to switch into, without forcing placeholder docs for the rest of `v4`.

## What ships in this first pass

- version-aware navigation, with `v3` as the current track and `v4` as the preview track
- the docs shell, theming, and page-level navigation you can keep filling out
- one preview page so version switching can be verified before the v4 content lands
