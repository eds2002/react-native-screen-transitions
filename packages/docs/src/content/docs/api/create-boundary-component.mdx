`createBoundaryComponent()` wraps a custom component with the boundary system.

```tsx
import Transition from "react-native-screen-transitions";

const BoundaryImage = Transition.createBoundaryComponent(Image, {
	alreadyAnimated: false,
});

<BoundaryImage id="avatar" source={avatar} style={styles.avatar} />;
```

The wrapped component keeps its original props and accepts boundary props such as `id`, `group`, `anchor`, `scaleMode`, `target`, `method`, and `enabled`.
